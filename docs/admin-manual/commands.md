# Commands

### Managing Users

#### List Users

`php artisan user:list`

#### Create a User

`php artisan user:create`

#### Change Password

`php artisan user:change-password`

#### Delete a User

`php artisan user:delete`

### Maintenance Mode

`php artisan down`

### Disable Maintenance Mode

`php artisan up`
